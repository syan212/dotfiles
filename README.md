# Dotfiles

My dotfiles

`update.sh`: quick shell script to update this repo.
`wallpapers/`: my directory for wallpapers. I keep it in my home directory, it currently only has one.
`bin/`: shell scripts I made. I put them in ~/bin.
`.config/nvim`: neovim configurations. I use lazy.nvim, with the following plugins:
- barbar.nvim
- cmp-buffer 
- cmp-cmdline
- cmp-nvim-lsp
- cmp-path
- cmp-vsnip
- gitsigns.nvim
- kanagawa.nvim
- lazy.nvim
- lualine.nvim
- nvim-cmp
- nvim-lspconfig
- nvim-tree.lua
- nvim-treesitter
- nvim-web-devicons
- vim-vsnip
`.config/kitty`: kitty config
`.config/paru`: paru config
`.zshrc`: my zsh config using oh-my-zsh
